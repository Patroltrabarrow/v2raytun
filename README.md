# v2RayTun — универсальный центр управления профилями Xray Core

**v2RayTun** — кроссплатформенный клиент для работы с конфигурациями подключений на базе **Xray Core**. Приложение доступно для **Android**, **Android TV**, **iOS**, **macOS**, **Windows** и **Linux**, предоставляя единое пространство для создания, хранения, настройки и использования пользовательских профилей.

Клиент поддерживает различные способы добавления конфигураций: импорт файлов настроек, загрузку профилей через подписочные ссылки, ручное создание параметров и изменение существующих записей. Все добавленные профили находятся под единым управлением, что позволяет быстро переключаться между различными вариантами подключения.

Благодаря общей архитектуре приложение обеспечивает одинаковый подход к работе с конфигурациями на любых устройствах — от мобильных платформ и телевизионных систем до настольных компьютеров.

> Официальный репозиторий **v2RayTun** содержит исходный код, готовые сборки, техническую документацию, сведения о поддерживаемых платформах, историю изменений и информацию о планах развития проекта.

## Использование приложения

> [!WARNING]
> **v2RayTun** является программным клиентом для **Xray Core** и не предоставляет собственные серверы, VPN-сервисы, прокси-доступ, подписки или готовые настройки подключения.

Для работы необходимо добавить собственную конфигурацию:

* импортировать файл профиля;
* загрузить настройки через ссылку подписки;
* создать параметры подключения вручную.

После добавления профилей пользователь может изменять их параметры, управлять списком конфигураций и выбирать активный вариант подключения.

## Поддерживаемые платформы и технологии

При создании **v2RayTun** используются технологии, соответствующие особенностям каждой операционной системы. Такой подход позволяет обеспечить стабильную работу приложения, оптимальное быстродействие и корректную интеграцию с возможностями конкретного устройства.

Некоторые функции могут внедряться постепенно, поскольку каждая платформа имеет собственные технические ограничения и особенности реализации.

| Платформа            | Технологический стек | Реализация                                                                |
| -------------------- | -------------------- | ------------------------------------------------------------------------- |
| Android / Android TV | Kotlin               | Адаптированная версия для смартфонов, планшетов и телевизионных устройств |
| Windows / Linux      | Flutter + Dart       | Кроссплатформенное приложение с отдельными параметрами сборки             |
| iOS                  | Swift                | Нативная реализация для экосистемы Apple                                  |
| macOS Apple Silicon  | Swift                | Оптимизация для устройств с ARM-чипами Apple                              |
| macOS Intel          | Swift                | Совместимость с компьютерами Mac на базе Intel                            |
